Página para subir fotografía.

Instrucciones:

Se recomienda solicitar ayuda a otra persona, para poder realizar este paso, ya que es necesario verificar el contenido y tamaño de la foto, de acuerdo a las especificaciones que se detallan mas adelante.

Para desplazarte dentro de las opciones puedes presionar la tecla tabulador, para utilizar un botón debes posicionarte sobre él y presionar la tecla enter.
